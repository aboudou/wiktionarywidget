Wiktionary Widget
=================

Based on Dashboard [Wikipedia widget](http://www.whatsinthehouse.com/widgets/), Wiktionary widget pulls up [Wiktionary](http://en.wiktionary.org) articles, the free and collaborative online dictionary.
